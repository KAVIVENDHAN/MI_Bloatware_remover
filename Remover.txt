Options: to combine with list packages command
	-k keep user data and cache
	-u got uninstalled
	-e enabled packages name
	-d disabled packages name
	-i For installed
	-s List only the system packages
	-3 To list only 3rd party (or non-system) packages
	-f List the package names as well as the path to the installed APK files. (an be combined with the other options as well)

Uninstall command:
--pm uninstall --user 0 PackageName
